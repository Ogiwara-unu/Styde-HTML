Tienda de calzado premium construida como SPA en vanilla JavaScript. Incluye catálogo de productos, carrito de compras, búsqueda en tiempo real y página de creadores con datos obtenidos desde la GitHub API.

## Estructura

```
PROYECTO2_RANDALL_LELANY/
├── css/styles.css      — Sistema de diseño unificado
├── js/script.js        — Lógica SPA: navegación, carrito, búsqueda
├── index.html          — Shell principal
└── creators.html       — Equipo con integración GitHub API
```

## Uso

Abrir `index.html` directamente en el navegador, o con servidor local para evitar restricciones CORS:

```bash
npx serve .
```
